⚠ Permission Error

The drive must be unmounted before scanning.

In Finder, right-click the Tesla drive → Eject, then click Scan again. The drive stays physically connected — macOS releases exclusive access.

Or run in Terminal: diskutil unmountDisk /dev/diskN

Raw device access requires elevated permissions.

Run with: sudo teslacam-recovery

Or add your user to the disk group, then log out and back in:
sudo usermod -aG disk $USER

Raw device access requires Administrator privileges.

Right-click TeslaCam Recovery → Run as administrator, then try again.
